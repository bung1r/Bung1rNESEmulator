# This is an NES (Nintendo Entertainment System) Emulator written in C#
## Made by bung1r! 

**This NES Emulator boasts:**
- An easy to use user interface for loading roms (along with some built in open source ROMs to test the emulator (i.e nestest.nes))
- Debugging tools built in to the rom (Pause, Game Speed, Show Pattern Tables + Palettes, etc.)
- 3 Mappers in order to play a majority of Official NES games (supports Super Mario, Donkey Kong, etc.)

**This NES Emulator will (probably) NOT boast:**
- Functionality for all NES games 
- Implementation of Illegal Opcodes (few official games rely on these anyway)
- Implementation of all Mappers (meaning some official games cannot be played)
